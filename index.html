O que é HTML?

HTML é a linguagem que usamos para estruturar o conteúdo de uma página web. Ele define títulos, parágrafos, imagens, links e muito mais.

Por que aprender HTML?

Sem HTML, não existe site! É o primeiro passo para qualquer desenvolvedor web.

Exemplo de Formulário

            
Para criar um formulário, usamos a tag <form>, e com ela podemos usar alguns atributos como por exemplo:

            
    
              
  • type = define qual vai ser o tipo de input
    • 
              
  • value = define o valor do campo
    • 
              
  • placeholder = define o valor do campo que fica "invisivel", enquanto o usuario não digita
    • 
            

            
Exemplo: 

            <form>
              <input type="text" name="txtnome" value="Digite seu nome">
            </form>







Exemplos de Destaque

Usando a tag <strong>, podemos dar mais importancia ao texto: Texto importante

Usando a tag <em>, podemos deixar o texto em itálico: Texto em destaque

Usando a tag <u>, podemos enfatizar algo: Sublinhado

Usando a tag <mark>, podemos usar para chamar a atenção: Marcador
