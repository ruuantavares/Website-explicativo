O que é Node.js?

Node.js é um ambiente de execução JavaScript do lado do servidor. Ele permite criar aplicações web, APIs, manipular arquivos e bancos de dados usando JavaScript.

Exemplo de servidor simples

Abaixo está um exemplo de um servidor Node.js que responde “Olá, mundo!”: 

          
          const http = require('http');

          const server = http.createServer((req, res) => {
            res.writeHead(200, {'Content-Type': 'text/plain'});
            res.end('Olá, mundo!');
          });

          server.listen(3000, () => {
            console.log('Servidor rodando em http://localhost:3000');
          });

Para rodar: abra o terminal na pasta do projeto e digite node server.js.

Exemplo com Express

Express é uma biblioteca popular para criar servidores web com Node.js: 

            
            const express = require('express');
            const app = express();
            const port = 3000;

            app.get('/', (req, res) => {
              res.send('Olá, Express!');
            });

            app.listen(port, () => {
              console.log(`Servidor rodando em http://localhost:${port}`);
            });

Instale o Express com npm install express.
