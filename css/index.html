O que é CSS?

CSS (cascading style Sheets) é a linguagem usada para estilizar paginas HTML. Com ela, podemos controlar as cores, tamanhos, espaçamentos,fontes e todo o layout da página.

Formas de aplicar o CSS


            -- Inline --
            <p style="color: blue;">Texto azul</p>

            -- Interno --
            <style>
              p { color: red; }
            </style>

            -- Externo --
            <link rel="stylesheet" href="estilo.css">

Principais Seletores

Seletor por TAG :


            p {
               color: blue;
              }

Aplica o estilo a todas as tags <p>

Seletor por classe :


          .texto {
                 color: green;
                }

Usado em elementos com class="texto"

Seletor por id :

          
          #titulo {
            font-size: 24px;
          }

Usado em elementos com id="titulo"

Seletores combinados :

          
          div p {
            color: purple;
          }

Afeta apenas <p> dentro de <div>

Propriedades de Texto :

          
          color: red; -- Cor do texto
          font-size: 20px; -- Tamanho da fonte 
          font-family: Arial, sans-serif; -- Tipo da fonte 
          font-weight: bold; -- Negrito 
          text-align: center; -- Alinhamento 
          text-decoration: underline; -- Sublinhado 
          line-height: 1.5; -- Espaçamento entre linhas

Propriedades de Caixa (Box Model) :

          
          width: 200px; -- Largura 
          height: 100px; -- Altura 
          margin: 10px; -- Espaço externo 
          padding: 15px; -- Espaço interno 
          border: 2px solid black; -- Borda 
          border-radius: 10px; -- Cantos arredondados 
          box-shadow: 2px 2px 5px gray; -- Sombra

Cores e fundos :

          
          background-color: lightblue; -- Cor de fundo 
          background-image: url('fundo.jpg'); -- Imagem de fundo 
          background-size: cover; -- Ajusta a imagem 
          opacity: 0.8; -- Transparência

Display e posicionamento :

          
          display: block; -- Ocupa toda a linha 
          display: inline; -- Fica na mesma linha 
          display: flex; -- Layout flexível 
          display: grid; -- Layout em grade 
          position: absolute; -- Posição fixa em relação ao pai 
          position: fixed; -- Fixa na tela 
          position: relative; -- Base para posições internas 
          top: 10px; left: 20px; -- Deslocamento

Flexbox (organização de layout) :

          
          display: flex;
          justify-content: center; -- Centraliza horizontalmente 
          align-items: center; -- Centraliza verticalmente 
          flex-direction: column; -- Empilha elementos 
          gap: 10px; -- Espaço entre itens

Grid (layout em colunas e linhas) :

          
          display: grid;
          grid-template-columns: 1fr 1fr 1fr; -- 3 colunas iguais 
          gap: 20px; -- Espaço entre os blocos

Outros úteis :

          
          cursor: pointer; -- Muda o cursor ao passar o mouse 
          transition: 0.3s; -- Animação suave 
          hover {
            background-color: yellow; -- Cor ao passar o mouse 
          }
          overflow: hidden; -- Oculta o que ultrapassa o bloco 
          z-index: 10; -- Define a ordem dos elementos
