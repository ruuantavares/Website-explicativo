O que é HTML?

HTML é a linguagem que usamos para estruturar o conteúdo de uma página web. Ele define títulos, parágrafos, imagens, links e muito mais.

Por que aprender HTML?

Sem HTML, não existe site! É o primeiro passo para qualquer desenvolvedor web.

Estrutura basica


          <!DOCTYPE html>
          <html> -- Define o início do documento HTML
          <head> -- Contém informações sobre o site 
              <meta charset="UTF-8"> -- Define o padrão de caracteres
              <title>Título da Página</title> -- Nome que aparece na aba do navegador
          </head>
          <body> -- Conteúdo visível da página 
          </body>
          </html>

Cabeçalhos e texto


          <h1> até <h6> - Títulos (H1 é o maior, H6 o menor) 
          <p> -- Parágrafo 
          <br> -- Quebra de linha 
          <hr> -- Linha horizontal 
          <strong> ou <b> -- Texto em negrito 
          <em> ou <i> -- Texto em itálico 
          <u> -- Sublinhado
